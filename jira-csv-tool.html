🖥️ CORS 프록시 실행 방법 (필수)

Jira API 연동을 사용하려면 로컬 CORS 프록시를 먼저 실행해야 합니다.

1단계: Node.js 설치 확인

터미널(CMD/PowerShell)에서 아래 명령어로 설치 여부 확인:

node -v

설치 안 되어 있다면 → https://nodejs.org 에서 LTS 버전 다운로드

2단계: 프록시 실행 파일 다운로드 & 실행

아래 버튼을 클릭하면 .bat 파일이 다운로드됩니다. 더블클릭으로 실행하세요.

또는 터미널에서 직접 실행:

npx local-cors-proxy --proxyUrl https://madngine.atlassian.net --port 8080

3단계: 프록시 실행 확인

Proxy Active 메시지가 뜨면 성공. 이 CMD 창은 닫지 마세요.

* 프록시는 이 도구를 사용하는 동안만 실행하면 됩니다. CMD 창을 닫으면 프록시도 종료됩니다.

📌 사용 방법

1. Jira 연결 — 위에서 도메인, 이메일, API 토큰을 입력하고 연결 테스트를 실행하세요.

2. 마일스톤 티켓 — 엑셀 기능 문서를 업로드하여 마일스톤 별 스토리 티켓을 일괄 생성합니다.

3. 이벤트/패키지 — 정기 업데이트 이벤트/패키지 스토리 티켓 6종 + 추가 피쳐를 생성합니다.

4. 하위 티켓 — 이벤트/패키지에서 생성한 스토리의 부서별 하위 작업 18종을 생성합니다.
